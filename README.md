# Guice-Demo

## In this project, we learn different ways for bindings:
- Linked Binding
- Binding Annotations
- Named Annotations
- Custom Binding
- Instance Binding
- Constructor Binding
- Built-in Binding
- Just-in-time Binding
- Untargeted Binding
- Injections:
  - Constructor Inject
  - Method Inject
  - Attribute Inject
- Binding life-cycle
  - Transient
  - Singleton
  - Scoped
